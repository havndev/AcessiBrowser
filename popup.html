Acerca

O AcessiBrowser foi criado como um projeto de faculdade para facilitar a acessibilidade num browser da web para pessoas com insuficiencias visuais.

A extensao conta com diferentes formas de facilitar este acesso a web, tais como uma aba que permite transformar o texto, modificando a fonte das letras, o seu tamanho, o espacamento entre letras e a sua cor.

Tambem possui uma aba onde e possivel transformar as cores da web, tal como aplicar filtros de cor ( Inverter Cores, Sepia,Grayscale), mas tambem permite mudar para modos que faciltiam a compreensao de conteudo para pessoas que sofram de daltonismo.

Por fim, tambem possui uma que implementa o ColorAdd, que permite traduzir cores para simbolos, de forma a aumentar a inclusao de todos os utilizadores da web.
