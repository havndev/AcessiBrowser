About

AcessiBrowser was created as a college project to enhance accessibility in a web browser for individuals with visual impairments.

The extension offers various ways to improve web accessibility, such as a tab that allows text transformation by changing the font, its size, letter spacing, and color.

It also has a tab where you can transform web colors, including applying color filters (Invert Colors, Sepia, Grayscale). It also allows switching to modes that enhance content comprehension for individuals with color blindness.

Finally, it also includes an implementation of ColorAdd, which translates colors into symbols to increase the inclusivity of all web users.
